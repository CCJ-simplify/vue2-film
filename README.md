# vue2-film

## 项目简介

> 此项目简单模仿[猫眼电影](https://m.maoyan.com)移动端的页面。简单实现了从'影片选择'到'生成订单'的功能。所有数据均为静态数据，并未实现从服务端获取数据功能。

## 技术栈

> vue2 + vuex + vue-router 


## 项目运行

``` bash

# 获取项目
git clone https://github.com/zd0423/vue2-film.git

# 安装依赖
npm install

# 开启本地服务器
npm run dev

# 发布环境
npm run build

```
